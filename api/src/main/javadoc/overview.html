Java API for JSON Processing API provides a way to process JSON.

This API can be used to:

  • Parse and generate JSON text in a streaming fashion (similar to StAX API for XML).
  • Build a Java object model (similar to DOM API for XML) for JSON text.

The streaming API is built upon JsonParser and JsonGenerator abstractions. A JsonParser can be used to parse JSON in a streaming way and a JsonGenerator can be used to generate JSON from scratch in a streaming way. It is a low-level API to process JSON and provides an efficient way to do the processing.

On the other hand, the object model API is a high-level API that provides object models for JSON object and array structures. These JSON structures are represented using JsonObject and JsonArray object models. The API uses builder patterns to build these object models of JSON from scratch. These object models can also be created from an input source using a JsonReader. Similarly, these object models can be written to an output source using a JsonWriter.
